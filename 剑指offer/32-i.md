## 思路

**层次遍历 --> BFS**

**BFS --> 队列(先入先出)**

**BFS 循环步骤：**

1. 初始化： 先将根节点 `root` 入队。

2. 出队： 队首元素出队， `queue.popleft()`。 在 python 中其实队列和栈的部分功能都可以用列表来写。

   举个 🌰： 比如 队列中的 `popleft()` 其实和列表的 `pop(0）` 功能相似。

3. 打印： 将当前节点值加入结果列表 `ans`

4. 入队： 当前节点如果有左右孩子的话， 依次入队。

## 代码
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def levelOrder(self, root: TreeNode) -> List[int]:
        if not root:
            return []
        queue = collections.deque()
        queue.append(root)
        ans = []
        while queue:
            cur = queue.popleft()
            ans.append(cur.val)
            if cur.left:
                queue.append(cur.left)
            if cur.right:
                queue.append(cur.right)
        return ans
```

## 复杂度分析 
- 时间复杂度： O(N)
- 空间复杂度： O(N)
