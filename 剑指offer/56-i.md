## 代码
```python
class Solution:
    def singleNumbers(self, nums: List[int]) -> List[int]:

        dic = collections.defaultdict(int)

        for num in nums:
            dic[num] += 1
        ans = []
        for key, value in dic.items():
            if dic[key] == 1:
                ans.append(key)
        return ans
```

## 复杂度分析
- 时间复杂度： O(n)
- 空间复杂度： O(n)
