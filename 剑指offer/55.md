## 题目描述

方法一： DFS
====
## 思路
疫苗第二针打完了， 烧了一天一夜我又复活了 ✌️。

来补前两天的 md 了。

DFS。

树的深度 = 左右子树的最大深度。

遍历方式： 后序遍历。

## 代码
```python
# Definition for a binary tree node.
# class TreeNode:
#     def __init__(self, x):
#         self.val = x
#         self.left = None
#         self.right = None

class Solution:
    def maxDepth(self, root: TreeNode) -> int:

        def dfs(root):
            if not root:
                return 0
            return 1 + max(dfs(root.left), dfs(root.right))
        return dfs(root)
```

## 复杂度分析
- 时间复杂度： O(n)
- 空间复杂度： O(n)

方法二： BFS
====
## 思路
BFS。

层序遍历。

将 queue 中的节点的左右节点， 也就是下一层的信息暂存在 level 中。

更新队列 queue = level 将下一层信息给 queue 。

此时更新层数。

## 代码
```python
class Solution:
    def maxDepth(self, root: TreeNode) -> int:
        if not root:
            return 0
        ans = 0
        queue = []
        queue.append(root)
        while queue:
            level = queue
            for node in level:
                cur = level.pop(0)
                if cur.left:
                    level.append(cur.left)
                if cur.right:
                    level.append(cur.right)
            queue = level
            ans += 1
        return ans
```

## 复杂度分析
- 时间复杂度： O(n)
- 空间复杂度： O(n)
